CTRL+ALT+T - команда открытия терминала
~ - символ домашней директории(папки)
/ - символ корневой директории(папки)
.. - обозначает предыдущую директорию
. - обозначает текущую директорию
CTRL+SHIFT+C - копировать в терминале
CTRL+SHIFT+V - вставить в терминале
1. ls - выводит список всех директорий и файлов в текущей директории(рабочая директория)
 ls -a -> выводит список как скрытых, так и обычных файлов и директорий
 ls -l -> выводит список директорий и файлов с подробной информацией
 ls -la
2. cd(change directory) - меняет текущую директорию на указанную
 cd  <название директории>
3. mkdir (make directory) - создает указанную директорию
 mkdir <директория> 
4. touch -  создает указанный файл
 touch <name_of_file>, <extension(расширение(txt, docs))>
5. pwd (print working directory) - выводит путь до текущей директории
6. rmdir (remove directory) - удаляет указанную директорию(не может удалить не пустую директорию)
7. rm - удаление файла
 rm <name_of_file>
 rm -rf - принудительное удаление файла, либо директории
8. mv(move) - перемещает, либо переименовывает файл/директорию в зависимости от указанного  второго аргумента
 mv <файл_либо_директория> <новое_имя -- директория/>
9. cat - выводит содержимое указанного файла
10. nano -  открывает графический интерфейс в терминале для редактирования файла
*если нет указанного файла, то создастся новый файл с таким именем
nano <name_of_file>
11. echo - команда для быстрой записи данных в файл через терминал
 echo текст для даписи > название файла
12. cp (copy) - команда для копирования файла в нужную директорию
 cp <file_to_copy> <path>
